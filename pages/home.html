Tools Overview

Compare Environments: This tool calculates checksums for each file in a folder and allows you to compare two folders to identify differences and missing files. You can navigate the comparison like browsing the original file structure.

Change Environments: Quickly switch CAD environments on your development machine to other customer environments and machines.

Cache Clearer: This tool is a Python application designed to help developers clean up cached files generated by Visual Studio and related project builds. It simplifies the process of removing bin, obj, and .vs folders from solution and project directories.
