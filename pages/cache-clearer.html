Cache Clearer

The Cache Clearer tool is a Python application designed to help developers clean up cached files generated by Visual Studio and related project builds. It simplifies the process of removing bin, obj, and .vs folders from solution and project directories.

Note: To clear all caches for all projects, you must close Visual Studio before running this tool. Failure to do so may result in permission errors, preventing some folders from being deleted.

How It Works

  1. Launch the Python application. A graphical user interface (GUI) will appear.
  2. Use the "Load .sln File" button to select a Visual Studio solution file (.sln).
  3. The application will list all projects within the solution along with their physical locations.
  4. To clear the cache:
    • Select specific projects or use the "Select All" button to select all projects.
    • Click the "Clear Entire Cache" button to delete the bin, obj, and .vs folders.
  5. The application will handle errors (e.g., permission issues) and continue processing other folders. At the end, it will display a summary of any folders that couldn't be deleted.

Features

  • Load .sln File: Lists all projects in the solution and their physical paths.
  • Clear Cache: Deletes bin, obj, and .vs folders from project directories and the .vs folder in the solution directory.
  • Selective Clearing: Allows clearing caches for selected projects or all projects at once.
  • Error Handling: Logs permission errors and continues processing other folders.

Best Practices

  • Close Visual Studio before running the tool to avoid permission errors.
  • Run the tool with sufficient permissions (e.g., as an administrator) if necessary.
  • Use the "Select All" option to ensure all project caches are cleared during major updates or cleanup tasks.

Example Workflow

      1. Launch the Cache Clearer tool.
      2. Click "Load .sln File" and select your solution file.
      3. Select all projects or specific projects to clear.
      4. Click "Clear Entire Cache."
      5. Review the summary to ensure all caches were successfully cleared.
